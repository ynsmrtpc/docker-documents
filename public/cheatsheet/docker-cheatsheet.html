Docker Temel Hızlı Referans

Docker Komutları

  • docker --version: Docker versiyonunu kontrol eder
  • docker info: Sistem genelinde bilgileri gösterir
  • docker help: Docker komutlarını listeler

Görüntülerle Çalışma

  • docker images: Tüm görüntüleri listeler
  • docker pull [görüntü_adı]: Bir görüntüyü indirir
  • docker rmi [görüntü_adı]: Bir görüntüyü kaldırır

Konteynerleri Yönetme

  • docker ps: Çalışan konteynerleri listeler
  • docker ps -a: Tüm konteynerleri listeler
  • docker run [görüntü_adı]: Bir konteyner çalıştırır
  • docker stop [konteyner_id]: Bir konteyneri durdurur
  • docker rm [konteyner_id]: Bir konteyneri kaldırır

Docker Ağları

  • docker network ls: Ağları listeler
  • docker network create [ağ_adı]: Bir ağ oluşturur
  • docker network rm [ağ_adı]: Bir aği kaldırır

Docker Depolama Alanları

  • docker volume ls: Depolama alanlarını listeler
  • docker volume create [depolama_alanı_adı]: Bir depolama alanı oluşturur
  • docker volume rm [depolama_alanı_adı]: Bir depolama alanını kaldırır

Docker Compose

  • docker-compose up: Hizmetleri başlatır
  • docker-compose down: Hizmetleri durdurur
  • docker-compose build: Hizmetleri oluşturur

Dockerfile

  • FROM: Temel görüntü
  • RUN: Yeni bir katmanda komutları çalıştırır
  • CMD: Varsayılan komutu ayarlar
  • COPY: Ana bilgisayardan konteynera dosyaları kopyalar
  • EXPOSE: Bir portu açar

Faydalı İpuçları

  • Kullanılmayan dosyaları dışlamak için .dockerignore kullanın
  • Görüntüleri sürüm numaralarıyla etiketleyin
  • Kullanılmayan görüntüleri ve konteynerleri düzenli olarak temizleyin
